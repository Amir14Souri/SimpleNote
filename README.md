# SimpleNote

**SimpleNote** is a mobile app for managing notes with an **offline-first** approach and secure authentication.

- **API docs (Redoc):** `https://simple-note.amirsalarsafaei.com/api/schema/redoc/`

---

## Features
- User signup & login  
- Paginated note list & search  
- View note details  
- Create, edit, delete notes  
- User profile & logout  
- Refresh-token authentication  
- Offline-first with local database
